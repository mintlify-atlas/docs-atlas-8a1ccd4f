Urbanshade OS ships with 81 applications across 9 categories. Some are built-in and always available; others are downloadable from the App Store.

<Info>
**32 built-in apps** are available immediately after installation (11 in Minimal, 15 additional in Standard). **49 downloadable apps** can be installed from the App Store at any time.
</Info>

## Opening apps

There are several ways to open an application:

- **Start menu** — Click the Start button (bottom-left) and browse or search for the app
- **Desktop icon** — Double-click any icon pinned to the desktop
- **Terminal** — Type the app's ID (e.g., `open terminal`) in the Terminal
- **App Store** — Browse and install downloadable apps, then launch from the Start menu

## Installing apps

<Steps>
  <Step title="Open the App Store">
    Click the Start menu and select **App Store**, or press `Shift+T` to open the Terminal and run `open app-store`.
  </Step>
  <Step title="Browse or search">
    Use the search bar or browse by category to find the app you want.
  </Step>
  <Step title="Install">
    Click **Install** on the app's listing. The installation runs in the background.
  </Step>
  <Step title="Launch">
    Find the app in the Start menu or on the desktop after installation completes.
  </Step>
</Steps>

<Tip>
Built-in apps are always available from the Start menu — no installation required. If you don't see a built-in app, check that your installation type included it, or reinstall via the App Store.
</Tip>
