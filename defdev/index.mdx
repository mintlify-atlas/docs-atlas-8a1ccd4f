## What is DEF-DEV?

DEF-DEV is the built-in developer console for Urbanshade OS. It provides low-level access to system internals — real-time logs, an interactive terminal, action monitoring, localStorage inspection, diagnostics, and administrative controls — without touching the production experience.

DEF-DEV 3.0 introduces a redesigned grouped navigation interface, integrated FakeMod testing, expanded terminal commands, and enhanced admin tooling.

<Note>
  DEF-DEV requires Developer Mode to be enabled. See [Setup](/defdev/setup) for all methods.
</Note>

## Quick start

<Steps>
  <Step title="Enable Dev Mode">
    Go to **Settings → Developer Options** and toggle **Enable Developer Mode** on.
  </Step>
  <Step title="Open the console">
    Press `Ctrl+Shift+D`, or click **Launch** in the Developer Options section.
  </Step>
  <Step title="Start debugging">
    Use the grouped navigation tabs on the right — **Core**, **Diagnostics**, **System**, **Tools** — to access each feature.
  </Step>
</Steps>

## Terminal commands

| Command | Description |
|---|---|
| `help` | Show all available commands |
| `clear` | Clear the terminal output |
| `status` | Display system status and diagnostics |
| `bugcheck <code>` | Trigger a test BSOD with code |
| `crash [type]` | Force a crash screen |
| `reboot` | Restart the system |
| `shutdown` | Power off the system |
| `ls [filter]` | List localStorage keys |
| `get <key>` | Get a storage value |
| `set <key> <value>` | Set a storage value |
| `del <key>` | Delete a storage key |
| `toast <type> <msg>` | Show a toast notification |
| `status` | Show system status |
| `queue` | Show pending commands |
| `lockdown [protocol]` | Trigger lockdown mode |
| `recovery` | Boot into recovery mode |

See the [Terminal reference](/defdev/terminal) for the full command set.

## FakeMod testing

<Warning>
  FakeMod actions are purely cosmetic and do not affect real user data or permissions.
</Warning>

FakeMod lets you test moderation UI without affecting real users. Trigger ban, warning, mute, and kick popups directly inside the DEF-DEV window for realistic testing.

| Action | Description |
|---|---|
| **Ban** | Full ban popup |
| **Warning** | Warning notification |
| **Mute** | Mute indicator |
| **Kick** | Session kick |

See [Admin Tools](/defdev/admin) for FakeMod usage details.
