Create applications, themes, terminal commands, and extensions. Everything you need to extend the Urbanshade OS experience.

## Tech stack

Urbanshade OS is built on:

- **React 18** — UI library with hooks
- **TypeScript** — type-safe JavaScript throughout
- **Vite** — fast build tool and dev server
- **Tailwind CSS** — utility-first CSS framework
- **Supabase** — auth, database, and edge functions
- **shadcn/ui** — headless component primitives

## Key files

| File | Description |
|------|-------------|
| `src/lib/appRegistry.tsx` | Central app definition factory |
| `src/lib/systemBus.ts` | Cross-component event bus |
| `src/components/WindowManager.tsx` | Window rendering switch |
| `src/lib/terminalScripts.ts` | Terminal script runner |
| `src/lib/persistence.ts` | localStorage state persistence |
| `src/types/window.ts` | `App` and `WindowData` interfaces |

## Quick start

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/aswdBatch/urbanshade-OS
    cd urbanshade-OS
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Start the dev server">
    ```bash
    npm run dev
    ```
    The app will be available at `http://localhost:5173`.
  </Step>
</Steps>
