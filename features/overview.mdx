## DEF-DEV console enhancements

<Info>
  DEF-DEV base was introduced in **v2.1**, made public in **v2.3**, and overhauled further in **v2.7**.
</Info>

Developer tools received significant upgrades with new debugging capabilities.

| Enhancement | Description |
|---|---|
| **Command search** | Search all available DEF-DEV commands with fuzzy matching and descriptions. |
| **Export / import config** | Save and restore DEF-DEV preferences including filters, theme, and custom settings. |
| **Terminal scripts** | Save multiple terminal commands as a script and execute them with a single command. |
| **Multi-tab terminal** | Dedicated tabs for Console, Actions, Storage, Terminal, and more. |
| **First-boot consent flow** | Explicit opt-in for DEF-DEV on first boot with alternative dev storage. |

See the full [DEF-DEV guide](/defdev/index) for the complete console reference.

## Crash & recovery system

<Info>
  Introduced in **v2.1** (BIOS/recovery) and expanded with severity levels in **v2.6**.
</Info>

Enhanced error handling and recovery options keep your system safe — and crashes more informative.

| Feature | Description |
|---|---|
| **Severity levels** | Errors carry a severity: WARNING (recoverable), CRITICAL (action needed), or FATAL (system halt). |
| **Crash dump files** | Download detailed `.dmp` files with full crash information for debugging and reporting. |
| **Stack trace capture** | Automatic stack trace logging to DEF-DEV on every bugcheck. |
| **Recovery mode** | Press `F2` during boot to access password reset, system repair, factory reset, and more. |

See the [BIOS & Recovery](/features/bios-recovery) page for the full recovery walkthrough and all BIOS settings.

## Sound effects

<Info>
  Introduced in **v2.6** as part of the Online Accounts & UX polish update.
</Info>

Immersive audio feedback for system events. Enable sound in **Settings → Sound**.

| Sound event | Trigger |
|---|---|
| Boot sounds | Plays during POST and OS startup |
| Notifications | Incoming alerts and system messages |
| Messages | New chat or inbox events |
| Success | Completed actions |
| Warnings | Non-critical alerts |
| Errors | Failed operations |
| Login / logout | Session start and end |
| App open / close | Application launch and exit |
