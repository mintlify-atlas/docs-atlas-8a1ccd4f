## Welcome to Urbanshade OS

Urbanshade OS is a browser-first operating system simulation built around a playful, developer-friendly desktop environment. It runs entirely in your browser — no installation required — and features a complete desktop experience with windows, a taskbar, start menu, 70+ applications, cloud-synced accounts, and an extensible package ecosystem.

## System status

Urbanshade OS is actively maintained and updated on a bi-weekly cycle (every 2nd Friday–Sunday).

- **Current version:** V3.5.0 "Polished" (build 8736, released 2026-03-09)
- **Live site:** [urbanshade.aswd.dev](https://urbanshade.aswd.dev)
- **GitHub:** [aswdBatch/urbanshade-OS](https://github.com/aswdBatch/urbanshade-OS)
- **Update cadence:** Every 2nd week, Friday–Sunday

<Note>
  Urbanshade OS stores all session data in your browser's `localStorage`. For persistent data across sessions and devices, create a free account and enable cloud sync in **Settings → Sync**.
</Note>
