Urbanshade OS has 22 shortcuts across 3 contexts: Boot, Desktop, and Terminal.

<Info>
Urbanshade OS uses **Shift** as its modifier key — not `Ctrl` or `Alt` — so desktop shortcuts never conflict with your browser or operating system. Boot shortcuts are the only exception: they use no modifier.
</Info>

## Boot shortcuts

These shortcuts work during the boot animation, before the login screen appears.

| Key | Action | Context |
|-----|--------|---------|
| `DEL` | Access BIOS settings | During boot |
| `F2` | Enter Recovery Mode | During boot |
| `F8` | Boot into Safe Mode | During boot |
| `ESC` | Skip boot animation | During boot |

<Tip>
If something is wrong with your installation, `F2` → Recovery Mode is your first stop. From there you can reset your password or do a factory reset.
</Tip>

## Desktop shortcuts

These shortcuts work any time you are on the desktop, as long as focus is not inside a text input field.

| Shortcut | Action |
|----------|--------|
| `Shift+/` | Toggle global search |
| `Shift+Tab` | Alt-Tab window switcher |
| `Shift+Esc` | Open Task Manager |
| `Shift+D` | Minimize all windows / Show desktop |
| `Shift+E` | Open File Explorer |
| `Shift+T` | Open Terminal |
| `Shift+I` | Open Settings |
| `Shift+L` | Lock screen |
| `Shift+Q` | Close focused window |
| `Shift+M` | Minimize focused window |
| `Shift+W` | Task view |
| `F11` | Toggle fullscreen |

<Warning>
If a shortcut is not working, check that you are not focused inside a text input or textarea. Desktop shortcuts are intentionally disabled in inputs to avoid interfering with typing.
</Warning>

## Terminal shortcuts

These shortcuts work while the Terminal app is open and focused.

| Key | Action |
|-----|--------|
| `↑` | Previous command in history |
| `↓` | Next command in history |
| `Tab` | Auto-complete command or path |
| `Ctrl+C` | Cancel current command |
| `Ctrl+L` | Clear terminal screen |
| `Ctrl+U` | Clear current input line |
