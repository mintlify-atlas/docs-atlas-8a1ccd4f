The Urbanshade OS terminal gives you direct, low-level control over the facility. Whether you're navigating the file system, monitoring system processes, or probing the network, every operation runs through a familiar shell interface. Type commands after the `$` prompt and press **Enter** to execute.

```bash
user@urbanshade:~$ _
```

> This is your terminal prompt. Type commands after the `$` symbol and press Enter.

## Basic commands

General-purpose commands for orientation and everyday use.

| Command | Args | Description |
| --- | --- | --- |
| `help` | | Show all available commands |
| `clear` | | Clear the terminal screen |
| `status` | | Display current system status |
| `whoami` | | Show current user identity |
| `date` | | Display current date and time |
| `echo` | `[text]` | Print text to the terminal |
| `history` | | Show command history |

## File system commands

Navigate directories and manage files.

| Command | Args | Description |
| --- | --- | --- |
| `ls` | | List files in current directory |
| `ls -la` | | List all files with details |
| `cd` | `[dir]` | Change directory |
| `cd ..` | | Go up one directory |
| `pwd` | | Print working directory path |
| `cat` | `[file]` | Display file contents |
| `mkdir` | `[name]` | Create a new directory |
| `rm` | `[file]` | Remove a file |
| `cp` | `[src] [dst]` | Copy a file |
| `mv` | `[src] [dst]` | Move or rename a file |

## System commands

Monitor and control processes, memory, disk, and power state.

| Command | Args | Description |
| --- | --- | --- |
| `neofetch` | | Show system information in style |
| `uptime` | | Display system uptime |
| `ps` | | List running processes |
| `kill` | `[pid]` | Terminate a process by ID |
| `top` | | Interactive process viewer |
| `free` | | Show memory usage |
| `df` | | Show disk space usage |
| `reboot` | | Restart the system |
| `shutdown` | | Power off the system |

## Network commands

Test connectivity, inspect interfaces, and transfer data.

| Command | Args | Description |
| --- | --- | --- |
| `ping` | `[host]` | Test network connectivity |
| `ifconfig` | | Show network interfaces |
| `netstat` | | Display network statistics |
| `curl` | `[url]` | Fetch content from a URL |
| `wget` | `[url]` | Download a file from a URL |

## Keyboard shortcuts

| Shortcut | Action |
| --- | --- |
| `↑` / `↓` | Navigate command history |
| `Tab` | Auto-complete commands and paths |
| `Ctrl+C` | Cancel the current command |
| `Ctrl+L` | Clear the screen |
| `Ctrl+U` | Clear the current line |

<Tip>
  Type `help` in the terminal for the most up-to-date list of all available commands. The terminal also supports tab completion — press **Tab** after typing a partial command or path.
</Tip>
